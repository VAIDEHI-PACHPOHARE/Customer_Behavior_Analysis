# Customer_Behavior_Analysis
Customer Shopping Behavior Analysis is an end-to-end Data Analytics project that analyzes customer purchasing patterns using Python, MySQL, and Power BI. The project includes data cleaning and preprocessing with Python (Pandas), SQL-based business analysis in MySQL, and an interactive Power BI dashboard to visualize key insights such as customer segmentation, revenue trends, product performance, subscription behavior, and sales analysis. This project demonstrates the complete data analytics workflow, from raw data processing to business intelligence reporting.
